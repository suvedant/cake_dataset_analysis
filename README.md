# 🎂 Cake Dataset Analysis

## 📌 Project Overview

This project focuses on analyzing a Cake Dataset using Python. The main objective is to explore the dataset, understand patterns, and visualize insights using data analysis techniques.

The project demonstrates skills in data cleaning, exploration, and visualization.

---

## 🎯 Objectives

* Load and explore the dataset
* Understand data structure and features
* Perform data cleaning and preprocessing
* Generate insights using visualizations

---

## 📊 Dataset Information

* Dataset Name: Cake Dataset
* Format: CSV
* Contains multiple attributes related to cakes (quantitative & qualitative data)

---

## 🛠️ Technologies Used

* Python 🐍
* Pandas (Data Analysis)
* Matplotlib (Data Visualization)
* Seaborn (Statistical Visualization)
* Jupyter Notebook

---

## 🔍 Steps Performed

### 1. Data Loading

* Loaded dataset using Pandas

### 2. Data Exploration

* Checked shape of dataset
* Viewed first and last rows
* Used `.info()` to understand structure

### 3. Data Cleaning

* Checked for missing values
* Verified data types

### 4. Data Visualization

* Created graphs using Matplotlib & Seaborn
* Visualized trends and patterns in dataset

---

## 📈 Key Insights

* Identified patterns in cake-related attributes
* Observed distribution of variables
* Compared different categories using graphs
